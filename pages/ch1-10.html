Task 10: nor_8

In this exercise you are required to implement an 8-input NOR gate following input, output and functionalities.

Inputs:

The inputs to the NOR gate are 8 bits wide and are named "a_0" through "a_7" as input [7:0] a.

Output:

The output of the NOR gate is a single bit and is named "y" as assign y = ~(|a).

Functionality:

A NOR gate outputs a logical 1 (or "true") only when all of its inputs are 0.

Testbench:

The testbench should instantiate your 8-input NOR gate with a certain name and verify that the output is correct. The testbench should initialize the inputs to random values and apply a sequence of input values to the design with certain delays to be able to see the changes over time. The expected output for each input sequence should be calculated and compared against the actual output.
